# learning_git
resources to help you learn git

Geting Started
- https://git-scm.com/book/en/v2/Git-Basics-Getting-a-Git-Repository

First Time
```
$ git add *.c
$ git add LICENSE
$ git commit -m 'initial project version'
```
Push changes back to master:
```
$ git clone repo-url
$ git add *
$ git commit -m "message"
$ git push
```
